# 重复逻辑

```text
找出项目中逻辑实质相同但被写了两遍或多遍的代码段。不是指模板代码，而是业务逻辑重复。

重点关注：

- 权限判断、状态流转判断重复
- 金额、热度、分数计算重复
- 参数校验、Redis key 拼接、callback 签名校验重复
- 错误码映射重复
- HTTP 与 WS payload 转换重复
- schema / model / response struct 字段映射重复
- 多处维护同一业务不变量

每组重复说明：

- 哪些文件包含重复逻辑
- 重复逻辑是什么
- 两处逻辑是否已经发生差异
- 差异会导致什么业务问题
- 建议合并方式

不要报告纯样板代码、生成代码或框架模板代码。
```
